HMTL의 장점

  1. 배우기 비교적 쉽다.
  2. 우리가 매일 사용하는 만큼 매우 중요하다.
  3. 저작권이 없다.

추천 무료 호스팅

웹서버를 직접 만들기 어렵기 때문에, 무료 서비스를 이용할 수 있다. 그 중에 github 라는 서비스를 이용할수 있지만, 검색엔진에 "free static web hostiong" 이라고 검색해서 더 다양한 서비스들을 이용할 수도 있다. 생활코딩에서는 github외의 다음의 몇가지 서비스를 추천한다.
  • bitballoon
  • neocities
  • amazon S3
  • Google Cloud Storage
  • Azure Blob
